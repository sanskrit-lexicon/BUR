Note from Jim to Odile:
=====================================================================
Another contributor has suggested some changes in Burnouf dictionary.
I would like to have your opinion on these.
The changes can be arranged in 7 Sections.
The attached file (prep2.txt) has examples of each of these sections.

This file may also be found at
https://github.com/sanskrit-lexicon/BUR/blob/master/issue3/prep2.txt.

section 1:  Should '5o' be changed to '5॰'  (and similarly for 1,2,...,9)
 Google translate does not recognize either of these forms.
 Another possibility is that '5o' should be changed to '5e'.
Section 2: Should '3e' be changed to '3ᵉ', and similarly for other digits.
 Google translate recognizes
   'le 23e ordre de saints' as 'the 23rd order of saints'
Section 3: Should '1re' be changed to '1ʳᵉ' ?
Section 4: Should '1er' be changed to '1ᵉʳ' ?
Section 5: Should '2d' be changed to '2ᵈ' ?

The next two do not involve numbers.
Section 6: Should 'ae' be changed to 'æ' ?
Section 7: Should 'AE' be changed to 'Æ' ?

Thanks for taking a look.

=====================================================================
Odile's response
=====================================================================

Section 1: Yes 5o should be changed into 5°‌, and same for other digits (and it should Not be changed into 5e as 5e means 5th and 5° means "in the fifth place"; same for the other digits)

Section 2: Yes of course 3e can be changed to '3ᵉ', and similarly for other digits. that is surely a better presentation.
Section 3: Should '1re' be changed to '1ʳᵉ' ? Yes, surely
Section 4: Should '1er' be changed to '1ᵉʳ' ? same, yes.
Section 5: Should '2d' be changed to '2ᵈ' ? same, yes.

About 2d i see in the example given that that abbreviation is not referenced, '2d' is for 'second'

Section 6: Should 'ae' be changed to 'æ' ? no, see below
Section 7: Should 'AE' be changed to 'Æ' ? yes
Yes, 'AE' should be changed to 'Æ'
but

First change all Jaena/Jaenas into Jaïna/Jaïnas
and all jaena/jaenas into jaïna/jaïnas.

-'kaer' (under cāru);

- zaez (under śaśa) (non latine word)
